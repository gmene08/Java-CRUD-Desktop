# Java-CRUD-Desktop
Projeto desktop desenvolvido em Java, com foco na prática de integração com o PostgreSQL e operações básicas de CRUD (Create, Read, Update, Delete). A aplicação foi estruturada seguindo o padrão MVC (Model-View-Controller), com ênfase nas boas práticas de código e organização em camadas.

O sistema oferece uma interface gráfica simples, permitindo o gerenciamento de dados e servindo como base para o entendimento de conceitos fundamentais como conexão com banco de dados, persistência de dados e boas práticas de desenvolvimento.
